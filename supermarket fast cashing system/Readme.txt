arduino.c file is Code for qr reader and post machine emulater,  who is based by arduino system.
This ardunio system have 3 * button and 1 * lcd 16x2 display. So you can enter qr code for products and
while finishing your shopping,this system waiting for your payment confirmation. You can use main program 
without ardunio.In customer interface, as you can see on this page is fullscreen and
dont have exit button. Therefore ,you have to exit with alt + f4 combination.
